# Reimagining Public Advertising: A Design Thinking Approach to  Visual Cleanliness

## Overview
This project examines visual pollution in public spaces through a design-thinking framework with emphasis on regulatory enforcement and governance mechanisms.

## Research Work
- Stakeholder research involving 50+ respondents
- Persona and POV framework development
- Identification of regulatory and enforcement gaps
- Structured problem definition aligned with municipal contexts

## Proposed Intervention
- QR-based permit verification model
- Technology-assisted enforcement mechanism integrating citizen reporting

## Academic Context
Developed as part of undergraduate coursework to explore governance-aware system design. Proposed interventions are conceptual and intended for research discussion.
